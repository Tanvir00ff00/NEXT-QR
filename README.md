# NEXT-QR

A professional, fully-customizable QR Code Generator built with **Next.js 14**, **Tailwind CSS**, and **TypeScript**.

## Features

-   **Live Preview**: Real-time updates as you type or customize.
-   **Advanced Styling**:
    -   Custom Dot Patterns (Square, Rounded, Classy, etc.)
    -   Corner Styling (Squares, Dots)
    -   Eye-catching Gradients & Colors
-   **Logo Integration**:
    -   Upload custom logos
    -   **Auto-Trim Whitespace**: Smart background removal
    -   **Smart Shapes**: Circle or Square masking
-   **Multiple Formats**: Export as PNG, SVG, JPEG, WEBP.
-   **High Resolution**: Support for 4k+ resolution downloads.
-   **Mobile Optimized**: Responsive "App-like" layout.

## Tech Stack

-   [Next.js](https://nextjs.org/)
-   [Tailwind CSS](https://tailwindcss.com/)
-   [Lucide React](https://lucide.dev/) (Icons)
-   `qr-code-styling` lib

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
